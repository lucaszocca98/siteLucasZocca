/* Habilidades */

HTML

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas da web e seus conteúdos. Como a base de qualquer página da web, HTML define a estrutura e a organização do conteúdo na internet. É a pedra angular do desenvolvimento web, essencial para qualquer pessoa que deseja criar e publicar conteúdo online.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para descrever a apresentação de documentos HTML. Ele permite controlar a aparência visual dos elementos em uma página da web, incluindo layouts, cores, fontes e espaçamento. CSS é fundamental para a criação de sites visualmente atraentes e responsivos, proporcionando uma experiência de usuário aprimorada e consistente em diferentes dispositivos e navegadores.

JavaScript

JavaScript é a linguagem de programação usada para adicionar interatividade e comportamento dinâmico a páginas da web. Permite criar conteúdos que respondem a ações do usuário, como cliques e movimentos do mouse, além de atualizar e manipular elementos HTML e CSS de forma dinâmica. JavaScript é essencial para o desenvolvimento web moderno, tornando os sites mais interativos, funcionais e envolventes.

Banco de dados e SQL

SQL (Structured Query Language) é a linguagem padrão usada para gerenciar e manipular bancos de dados relacionais. Ele permite criar, modificar e consultar bancos de dados, além de inserir, atualizar e excluir dados. SQL é fundamental para o desenvolvimento de aplicações que requerem armazenamento e recuperação eficientes de grandes volumes de dados, sendo uma habilidade essencial para administradores de banco de dados e desenvolvedores de software.

Power BI

Power BI é uma ferramenta de análise de negócios desenvolvida pela Microsoft, usada para visualizar dados e compartilhar insights em toda a organização. Com ela, é possível conectar-se a diversas fontes de dados, criar relatórios interativos e painéis personalizados, facilitando a tomada de decisões baseadas em dados. Power BI é essencial para profissionais que desejam transformar dados brutos em informações valiosas e acessíveis, promovendo uma cultura orientada por dados.
